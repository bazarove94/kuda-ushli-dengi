# Куда ушли деньги?

Локальное и семейное приложение для учета доходов, расходов, лимитов, целей и чеков.

## Запуск

Откройте `Куда ушли деньги.html` двойным кликом. Сервер для локального режима не требуется.

## Семейная синхронизация

1. Создайте проект Supabase.
2. Выполните целиком файл `supabase-family.sql` в Supabase SQL Editor.
3. Укажите в HTML `SUPABASE_URL` и frontend `SUPABASE_ANON_KEY`.
4. В приложении откройте `Настройки` → `Семейный доступ`.
5. Зарегистрируйтесь и создайте семейную группу.
6. Передайте код `FAMILY-XXXXXX` второму участнику.
7. Второй участник регистрируется и подключается по этому коду.

Пароли обрабатывает только Supabase Auth. Во frontend нельзя использовать `service_role`, пароль базы данных и другие секретные ключи.

Realtime включен для:

- `transactions`;
- `category_limits`;
- `beneficiary_limits`;
- `goals`;
- `family_settings`.

Если Realtime временно недоступен, нажмите `Синхронизировать` в настройках.

## Работа без сети

В облачном режиме несинхронизированные изменения сохраняются в отдельной pending-очереди текущей семьи. После восстановления подключения приложение автоматически отправляет их в Supabase. Полная семейная база после выхода из аккаунта локально не отображается.

## Чеки

Фото и QR-код чека обрабатываются в браузере. Изображение не сохраняется. QR используется для суммы, даты, времени и фискальных реквизитов.
